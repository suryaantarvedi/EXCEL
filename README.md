# 📊 Excel Portfolio

This repository contains real-world Excel projects showcasing dashboard design, data modeling, and interactive analytics.  

###  Call Centre Performance Dashboard 

An interactive dashboard project built in Excel to analyze call centre operations.  

It includes:  
- 🎨 **Design & Layout**: Custom color theme, fonts, and recruiter-ready formatting  
- 🗄️ **Data Model**: Connecting multiple datasets with relationships  
- 📊 **KPIs & Measures**: Using Power Pivot & DAX for metrics like Calls, Amount, Avg Rating, Happy Callers  
- 🔍 **Pivot Tables & Charts**:  
   - Monthly call trend line graph  
   - Weekly distribution bar chart  
   - Calls & revenue by representative  
   - Rating distribution histogram  
   - Customer revenue analysis table  
- 🎛️ **Interactivity**: Slicers for filtering, conditional formatting for emphasis  
- 🖼️ **Dynamic Images**: XLOOKUP used to display representative photos
